# NEURYNTH ERP: Human Resources (HR) Domain

Управление самым ценным активом компании — людьми. Домен охватывает весь жизненный цикл сотрудника.

## Состав модулей
- **`neurynth_hr`**: Личные дела сотрудников, иерархия и контракты.

## Стратегия (Google Style)
- **Modular Data**: Данные о сотрудниках доступны другим модулям только через авторизованные вызовы.
- **Scalability**: Базовая структура готова к поддержке тысяч записей.
